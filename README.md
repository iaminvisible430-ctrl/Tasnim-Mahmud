## 🚀 Deploy to Vercel via GitHub (Recommended)

### Step 1 — Push to GitHub

```bash
# 1. Create a new repo on github.com (name it: tasnim-mahmud-hq)
# 2. Then run these commands in this folder:

git init
git add .
git commit -m "🚀 Initial commit — Tasnim Mahmud AI Founder HQ"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/tasnim-mahmud-hq.git
git push -u origin main
```

### Step 2 — Connect to Vercel

1. Go to **[vercel.com](https://vercel.com)** → Login with GitHub
2. Click **"Add New Project"**
3. Select your **`tasnim-mahmud-hq`** repository
4. Vercel will auto-detect the config from `vercel.json`
5. Click **"Deploy"** — Done! ✅

### Step 3 — Custom Domain (Optional)

In Vercel dashboard → Your project → **Settings → Domains**  
Add your domain e.g. `tasnimahmud.com` or `mahmud.dev`
